# Project Rules

## Language
- All code, comments, file names, variables, UI text, and content: English only.
- Never use Turkish in code or files.

## Token Efficiency
- Be concise. No long explanations unless I ask.
- Do not summarize what you just did unless I ask.
- Never read node_modules, .next, .vercel, or build outputs.
- Do not run npm install, npm run build, npm run dev, or test commands unless I explicitly ask.
- Do not run git commands unless I explicitly ask.

## Before Acting
- Before creating, modifying, or deleting any file: ask me first with a one-line summary.
- Before running any shell command: ask me first.
- If unclear, ask one short clarifying question instead of guessing.

## Communication
- Default to short answers.
- When done, just say "done" plus the minimum needed info.
- Do not repeat my instructions back to me.

## Project Context
- Stack: Next.js 15, TypeScript, Tailwind CSS, App Router.
- Goal: a public web app for AI-powered university physics learning.
- Database: Postgres (Neon) via Prisma — to be added later.
- Hosting: Vercel.
